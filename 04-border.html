Borders

The border property allows you to specify the style, width, and color of an element's border

Border Styles

The border-style property can take up to four values (top border, right border, bottom border, left border). The following border-style values are allowed:

  • dotted: dotted line border
  • dashed: dashed line border
  • solid: solid line border
  • double: double line border
  • groove: 3D grooved border, depends on border-color value
  • ridge: 3D ridge border, depends on border-color value
  • inset: 3D inset border, depends on border-color value
  • none: no border
  • hidden: hidden border (we cannot see it)
  • mixed: when we set multiple different borders for each side

Border Width

VERY IMPORTANT: border-width will not work unless you have a border-style set!!

border-width can be set to a specific size (px, em, etc.) or you can use one of the following settings:

  • thin
  • medium
  • thick

The border-width property can also take up to 4 inputs (top, right, bottom, left)

Border Color

VERY IMPORTANT: border-color will not work unless you have a border-style set!!!

border-color can be set the following ways:

  • name: specify a color such as "rebeccapurple"
  • hex: specity a hex value, such as "#6fca4b"
  • rgb: specify an rgb color, such as "rgb(202, 101, 50)"
  • rgba: specify an rgba color, such as "rgba(22, 22, 22, 0.5)"
  • transparent
