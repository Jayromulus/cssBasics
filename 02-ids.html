CSS IDs

an ID is a UNIQUE identifier

ID Notes

  • In CSS, an id starts with an octothorpe (pound, hashtag, number sign, etc.) - #
  • Example use for IDs: targeting a specific header or an about section
  • Each ID should only be used once per page - they are unique
    This will be important in JavaScript Later
  • Used for one-off styling
