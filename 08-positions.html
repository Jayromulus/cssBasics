Positioning

the position property specifies the type of positioning method used for an element

The four positioning values are:

  • fixed: this will pin something to a spot and it will stay there as a user scrolls. (ex: nav bar, footer) Fixed position takes an element out of the document
  • static: this is the default positioning in html. It is not affected by top, bottom, left, right properties. It is always positioned according to the normal flow of the page.
  • relative: does not take the element out of the normal document flow. this does, however, allow us to move it using the top, right, bottom, and left properties. Other content will not be adjusted to fill gaps left by a relative element
  • absolute: takes the element out of the normal document flow. if the parent is set to relative, and the child is set to absolute, the absolute position of 0,0 is going to be the top left corner of the parent element

The z-index property allows you to override the normal stacking order of tthe page. the default z-index is 0; and you can set that value to be positive or negative. An element with a greater z-index will stack in front of an element with a lower z-index. if elements have the same z-index, the element which appears later on the html document will be show on top. you must have a position assigned in order to use the z-index property.
