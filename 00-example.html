CSS Statements

CSS statements are made up of four main components:

  • Declaration: the statement as a whole with the curly braces
  • Selector: the name of the thing you want to style
  • Property: properties of the element to edit (height, width, color, etc.)
  • Value: value(s) we assign to the property (number, color, url, etc.)
