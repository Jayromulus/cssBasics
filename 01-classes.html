Classes

A CSS class is an identifier of an element which can be used multiple times on a page

Memory Device: Classes in school are also called "periods", when referring to a class in css, we MUST use a . before the class name

  • Classes start with a . in the selector .my-class-name
  • The same class name can be used more than once in an html document
  • The class keyword refers to the class name you set in the html page
